# Java Snake Game 🐍

A real-time Snake Game built using Java Swing and AWT.

## Features
- Dynamic snake growth
- Collision detection (walls & self)
- Score tracking
- Restart functionality (Press TAB)

## Controls
- Arrow Keys → Move Snake
- TAB → Restart Game

## How to Run
1. Compile:
   javac SnakeGame.java

2. Run:
   java SnakeGame
